# The React Quiz

This project is a quiz application built with React. It features a timer, scoring system, and various states to manage
the quiz flow.

## Technologies Used

- JavaScript
- React
- json-server

## How to Run

1. Install dependencies:
    ```bash
    npm install
    ```
2. Start the JSON server:
    ```bash
    npm run server
    ```
3. Start the development server:
    ```bash
    npm start
    ```
